# dn-background-jobs
==================================
- Body Parsing via [body-parser](https://github.com/expressjs/body-parser)

Getting Started
---------------

```sh
# clone it
git clone git@github.com:lukyr/db-bcakground-jobs.git
cd db-bcakground-jobs

# create folder data
mkdir data

# Install dependencies
npm install

# run docker compose
docker-compose up -d

#log container
docker-compose logs -f -t <service-name>
```
